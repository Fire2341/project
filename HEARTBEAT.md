# HEARTBEAT.md - 心跳检查清单

**最后更新**：2026-03-12  
**用途**：每次心跳时自动执行检查

---

## 检查清单

### 1. 当日记录检查
- [ ] `memory/YYYY-MM-DD.md` 是否存在且已更新
- [ ] 最后记录时间是否在最近10分钟内
- [ ] 用户发送的消息数 vs 已归档消息数是否匹配

### 2. 分类归档检查
- [ ] T0-T4分类是否完成
- [ ] 是否有标记【待确认】的分类

### 3. GitHub备份检查
- [ ] 是否有未提交的变更
- [ ] 最后一次提交时间

---

## 执行流程

```
收到心跳
    ↓
执行上述检查清单
    ↓
发现遗漏 → 立即补录/处理
    ↓
无遗漏 → 回复 HEARTBEAT_OK
```

**注意**：此文件在每次心跳时自动读取执行。
