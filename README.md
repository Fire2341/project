# OpenClaw Workspace

个人工作空间仓库，用于归档每日记录、投研笔记、项目文档等。

## 目录结构

- `memory/` - 每日原始记录
- `T0/` - 工作方法论
- `T1/` - 投研系统
- `T2/` - 个人成长/健康
- `T3/` - 项目/能力扩展
- `T4/` - 个人IP运营
- `screenshots/` - 截图存档
- `links/` - 链接存档

## 核心文件

- `AGENTS.md` - 助手行为规范
- `SOUL.md` - 人格定义
- `USER.md` - 用户画像
- `MEMORY.md` - 长期记忆
- `HEARTBEAT.md` - 心跳检查清单
- `TOOLS.md` - 本地工具配置
